# Контроллер поверхностей управления Як-130 - Документация

## Введение

Это комплексное приложение для управления моделью самолёта Як-130 на базе Arduino. Приложение предоставляет:

- **Управление 4 сервоприводами** для поверхностей управления
- **Расчёт аэродинамических характеристик** полёта
- **Удобный GUI** на PyQt5
- **Управление COM-портами** и диагностика соединения

## Архитектура

### Компоненты системы

#### 1. Arduino (servo_controller.ino)
Микроконтроллер управляет четырьмя сервоприводами:
- **Элероны (AIL)** - вращение крыла (крен)
- **Закрылки (FLP)** - интенсивность подъёмной силы
- **Руль направления (RUD)** - разворот вокруг вертикальной оси
- **Руль высоты (ELV)** - тангаж (вверх-вниз)

**Команды Serial:**
```
AIL:angle     - Установить угол элеронов (0-180)
FLP:angle     - Установить угол закрылок (0-180)
RUD:angle     - Установить угол руля направления (0-180)
ELV:angle     - Установить угол руля высоты (0-180)
STATUS        - Получить статус всех сервоприводов
RESET         - Вернуть все сервоприводы в центр
```

#### 2. Python приложение

##### main.py
- Главное окно и управление UI
- Интеграция всех компонентов
- Обработка пользовательских действий

##### gui.py
- **ControlPanel** - управление сервоприводами (слайдеры, спинбоксы)
- **CharacteristicsPanel** - отображение расчётных параметров
- **ConnectionPanel** - управление COM-портом

##### aircraft_calculator.py
Расчёт аэродинамических характеристик:
- Плотность воздуха на высоте (ISA модель)
- Статическое давление
- Температура воздуха
- Число Маха
- Динамическое давление
- Подъёмная сила и сопротивление
- Коэффициенты CL и CD
- Перегрузка (G-force)
- Скорость разворота
- Вертикальная скорость

##### com_port_manager.py
Управление Serial соединением:
- Список доступных портов
- Подключение/отключение
- Отправка команд
- Чтение ответов
- Тестирование соединения

##### servo_controller.py
Высокоуровневое управление сервоприводами:
- Установка позиций с плавным движением
- Предопределённые манёвры
- Получение текущих позиций
- Симуляция полёта

## Установка и запуск

### 1. Подготовка Arduino

1. Загрузить `arduino/servo_controller.ino` на микроконтроллер
2. Подключить сервоприводы к пинам:
   - Элероны: GPIO 9
   - Закрылки: GPIO 10
   - Руль направления: GPIO 11
   - Руль высоты: GPIO 12

### 2. Установка Python

```bash
# Установить зависимости
pip install -r requirements.txt

# Запустить приложение
python src/main.py
```

## Использование

### Вкладка "Управление"

Прямое управление каждым сервоприводом:
- Слайдер для быстрого изменения
- Спинбокс для точного ввода
- Отображение текущего угла

### Вкладка "Соединение"

1. Выбрать COM-порт из списка
2. Нажать "Обновить" для поиска портов
3. Нажать "Подключиться"
4. Нажать "Тестировать" для проверки

### Вкладка "Манёвры"

Выполнять предопределённые манёвры:
- Разворот влево/вправо
- Набор высоты/снижение
- Рыкание влево/вправо
- Взлёт/посадка

### Вкладка "Предустановки"

Быстрые предустановки режимов полёта:
- Горизонтальный полёт
- Набор высоты
- Снижение
- Крейсерский режим

### Параметры полёта (справа)

Установить параметры для расчётов:
- **Скорость** - 0-300 м/с
- **Высота** - 0-15000 м
- **Угол атаки** - -20 до +20°
- **Крен** - -45 до +45°
- **Тангаж** - -30 до +30°
- **Дроссель** - 0-100%

После установки параметров:
1. Система автоматически рассчитывает характеристики
2. Нажать "Применить рассчитанные команды" для отклонения рулей

### Характеристики (справа вверху)

Отображаются рассчитанные параметры:

**Параметры полёта:**
- Скорость полёта
- Высота
- Угол атаки
- Крен, тангаж
- Число Маха

**Аэродинамика:**
- Динамическое давление
- Подъёмная сила и сопротивление
- Коэффициенты Cy (CL) и Cx (CD)
- Перегрузка (G-force)

**Атмосфера:**
- Плотность воздуха
- Давление и температура
- Скорость разворота
- Вертикальная скорость
- Доступная тяга

## Расчётные модели

### Аэродинамика

Используются упрощённые аэродинамические модели для расчёта:

```
L = 0.5 * ρ * V² * S * CL
D = 0.5 * ρ * V² * S * CD
Q = 0.5 * ρ * V²  (динамическое давление)
```

### Атмосфера (ISA модель)

Расчёт параметров атмосферы на основе Международной Стандартной Атмосферы:
- Температура: T = T₀ - L*h (до 11 км)
- Давление: P = P₀ * (T/T₀)^(-g*M/(R*L))
- Плотность: ρ = ρ₀ * (T/T₀)^(-(g*M/(R*L)+1))

### Перегрузки (G-force)

```
G = sqrt((V²/(g*R_pitch))² + (1/cos(bank))²)
```

## Возможные проблемы

### COM-порт не найден

- Проверить, подключено ли Arduino
- Установить драйверы CH340 или FTDI
- Проверить кабель USB

### Сервоприводы не отвечают

- Проверить соединение сервоприводов с Arduino
- Убедиться, что напряжение питания достаточно
- Перезагрузить Arduino (кнопка Reset)

### Ошибка "CONNECTION TIMEOUT"

- Проверить скорость передачи (9600 baud)
- Проверить кабель (может быть неисправен)
- Перезагрузить приложение

### Странные расчёты характеристик

- Проверить параметры Як-130 в `config/aircraft_params.json`
- Убедиться, что параметры полёта в разумных пределах

## Расширение функционала

### Добавление нового манёвра

Отредактируйте `src/servo_controller.py`, метод `create_maneuver()`:

```python
maneuvers = {
    'my_maneuver': {
        'ailerons': 80,
        'rudder': 100,
        'elevator': 95,
        'flaps': 10
    }
}
```

### Изменение расчётных модёлей

Отредактируйте методы в `src/aircraft_calculator.py`:
- `estimate_cl_coefficient()` - для изменения подъёмной силы
- `estimate_cd_coefficient()` - для изменения сопротивления
- `calculate_servo_commands()` - для изменения алгоритма управления
